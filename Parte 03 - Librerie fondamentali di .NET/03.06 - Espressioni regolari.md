# Esempio completo: importare righe con Regex

Scenario:

```text id="y8arcv"
Abbiamo un file di righe testuali.
Ogni riga valida ha formato:
CLIENTE;<id>;<nome>;<email opzionale>

Esempi:
CLIENTE;1;Emilie Rollandin;emilie@example.com
CLIENTE;2;Mario Rossi;
```

Regex:

```text id="mh7zpb"
^CLIENTE;(?<id>\d+);(?<nome>[^;]+);(?<email>[^;]*)$
```

Codice:

```csharp id="a6qzus"
using System.Text.RegularExpressions;

public record ClienteImportDto(
    int Id,
    string Nome,
    string? Email);

public static partial class ClienteImportParser
{
    [GeneratedRegex(@"^CLIENTE;(?<id>\d+);(?<nome>[^;]+);(?<email>[^;]*)$")]
    private static partial Regex RigaClienteRegex();

    public static ClienteImportDto? ProvaLeggiRiga(string riga)
    {
        Match match = RigaClienteRegex().Match(riga);

        if (!match.Success)
        {
            return null;
        }

        int id = int.Parse(match.Groups["id"].Value);
        string nome = match.Groups["nome"].Value.Trim();
        string emailTesto = match.Groups["email"].Value.Trim();

        string? email = string.IsNullOrWhiteSpace(emailTesto)
            ? null
            : emailTesto;

        return new ClienteImportDto(id, nome, email);
    }
}
```

Uso:

```csharp id="m4l833"

[
    "CLIENTE;1;Emilie Rollandin;emilie@example.com",
    "CLIENTE;2;Mario Rossi;",
    "RIGA NON VALIDA"
];

foreach (string riga in righe)
{
    ClienteImportDto? cliente = ClienteImportParser.ProvaLeggiRiga(riga);

    if (cliente is null)
    {
        Console.WriteLine($"Riga non valida: {riga}");
        continue;
    }

    Console.WriteLine($"{cliente.Id}: {cliente.Nome} - {cliente.Email ?? "nessuna email"}");
}
```

Output:

```text id="f9u594"

2: Mario Rossi - nessuna email
Riga non valida: RIGA NON VALIDA
```

# Esempio completo: normalizzare telefono

Scenario:

```text id="na7r9k"
L‚Äôutente pu√≤ scrivere un numero con spazi, trattini, parentesi.
Vogliamo tenere solo cifre e, opzionalmente, il + iniziale.
```

```csharp id="60bajo"
using System.Text.RegularExpressions;

public static partial class TelefonoNormalizer
{
    [GeneratedRegex(@"[^\d+]")]
    private static partial Regex CaratteriNonAmmessiRegex();

    [GeneratedRegex(@"\+(?=.)")]
    private static partial Regex PiuNonInizialeRegex();

    public static string Normalizza(string telefono)
    {
        ArgumentException.ThrowIfNullOrWhiteSpace(telefono);

        string pulito = CaratteriNonAmmessiRegex().Replace(telefono, "");

        if (pulito.StartsWith('+'))
        {
            string resto = pulito[1..].Replace("+", "");
            return "+" + resto;
        }

        return pulito.Replace("+", "");
    }
}
```


# Linee guida pratiche

```text id="4lcd32"
Usa Regex
  per pattern testuali, estrazione, sostituzione e validazione semplice.

Usa ^ e $
  quando vuoi validare tutta la stringa.

Usa gruppi nominati
  quando estrai parti significative.

  per rendere leggibili i backslash.

  se vengono usate molte volte.

  per pattern noti a compile-time.

Imposta timeout
  per input grandi o non fidati.

Non usare Regex
  per JSON, XML, HTML arbitrario o formati complessi con parser dedicati.
```
