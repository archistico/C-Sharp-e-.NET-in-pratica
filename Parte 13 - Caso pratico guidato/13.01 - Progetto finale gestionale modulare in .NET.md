# AppDbContext

```csharp
public class AppDbContext : DbContext
{
    public AppDbContext(DbContextOptions<AppDbContext> options)
        : base(options)
    {
    }

    public DbSet<Libro> Libri => Set<Libro>();

    public DbSet<Autore> Autori => Set<Autore>();

    public DbSet<Collana> Collane => Set<Collana>();

    public DbSet<Cliente> Clienti => Set<Cliente>();

    public DbSet<Documento> Documenti => Set<Documento>();

    public DbSet<DocumentoRiga> DocumentoRighe => Set<DocumentoRiga>();

    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        modelBuilder.Entity<LibroAutore>()
            .HasKey(x => new { x.LibroId, x.AutoreId });

        modelBuilder.Entity<Libro>()
            .Property(x => x.Prezzo)
            .HasPrecision(10, 2);

        modelBuilder.Entity<Documento>()
            .Property(x => x.TotaleDocumento)
            .HasPrecision(10, 2);

        modelBuilder.Entity<DocumentoRiga>()
            .Property(x => x.PrezzoUnitario)
            .HasPrecision(10, 2);

        modelBuilder.Entity<DocumentoRiga>()
            .Property(x => x.ScontoPercentuale)
            .HasPrecision(5, 2);

        modelBuilder.Entity<DocumentoRiga>()
            .Property(x => x.TotaleRiga)
            .HasPrecision(10, 2);
    }
}
```

Nota didattica importante:

```text
Anche se SQLite non applica precisione decimal come SQL Server,
mantenere la configurazione nel modello è utile per chiarezza
e per eventuale migrazione futura verso altri database.
```

# Minimal API

Useremo Minimal API, ma senza mettere tutto in `Program.cs`.

Errore da evitare:

```text
Program.cs enorme con 200 endpoint
```

Meglio dividere per moduli.

Esempio:

```csharp
public static class CatalogoEndpoints
{
    public static IEndpointRouteBuilder MapCatalogoEndpoints(
        this IEndpointRouteBuilder app)
    {
        RouteGroupBuilder group = app.MapGroup("/api/catalogo")
            .WithTags("Catalogo");

        group.MapGet("/libri", GetLibriAsync);
        group.MapGet("/libri/{id:int}", GetLibroAsync);
        group.MapPost("/libri", CreateLibroAsync);
        group.MapPut("/libri/{id:int}", UpdateLibroAsync);
        group.MapDelete("/libri/{id:int}", DeleteLibroAsync);

        return app;
    }

    private static async Task<IResult> GetLibriAsync(
        AppDbContext dbContext,
        CancellationToken cancellationToken)
    {
        List<LibroListItemDto> libri = await dbContext.Libri
            .AsNoTracking()
            .OrderBy(x => x.Titolo)
            .Select(x => new LibroListItemDto(
                x.Id,
                x.Titolo,
                x.Isbn,
                x.Prezzo,
                x.Attivo))
            .ToListAsync(cancellationToken);

        return Results.Ok(libri);
    }
}
```

In `Program.cs`:

```csharp
app.MapCatalogoEndpoints();
app.MapClientiEndpoints();
app.MapDocumentiEndpoints();
app.MapDashboardEndpoints();
app.MapReportEndpoints();
```

# Servizi applicativi

Per alcune operazioni CRUD semplici, l’endpoint può usare direttamente `AppDbContext`.

Per logiche più complesse, useremo servizi.

Esempio: calcolo documento.

```csharp
public interface IDocumentoCalculator
{
    DocumentoTotals CalcolaTotali(
        IReadOnlyList<DocumentoRigaInput> righe);
}
```

```csharp
public record DocumentoRigaInput(
    int Quantita,
    decimal PrezzoUnitario,
    decimal ScontoPercentuale);
```

```csharp
public record DocumentoTotals(
    decimal TotaleImponibile,
    decimal TotaleSconto,
    decimal TotaleDocumento);
```

Implementazione:

```csharp
public class DocumentoCalculator : IDocumentoCalculator
{
    public DocumentoTotals CalcolaTotali(
        IReadOnlyList<DocumentoRigaInput> righe)
    {
        decimal totaleLordo = 0;
        decimal totaleDocumento = 0;

        foreach (DocumentoRigaInput riga in righe)
        {
            decimal lordo = riga.Quantita * riga.PrezzoUnitario;

            decimal netto = lordo *
                (1 - riga.ScontoPercentuale / 100);

            totaleLordo += lordo;
            totaleDocumento += netto;
        }

        decimal totaleSconto = totaleLordo - totaleDocumento;

        return new DocumentoTotals(
            TotaleImponibile: totaleLordo,
            TotaleSconto: totaleSconto,
            TotaleDocumento: totaleDocumento);
    }
}
```

Questo servizio è perfetto per un test unitario.

# Esempio pagina Blazor: elenco libri

```razor
@page "/libri"
@inject CatalogoApiClient CatalogoApiClient

<h1>Libri</h1>

@if (isLoading)
{
    <p>Caricamento...</p>
}
else if (errorMessage is not null)
{
    <p class="text-danger">@errorMessage</p>
}
else
{
    <table class="table">
        <thead>
            <tr>
                <th>Titolo</th>
                <th>ISBN</th>
                <th>Prezzo</th>
                <th>Attivo</th>
            </tr>
        </thead>
        <tbody>
            @foreach (LibroListItemDto libro in libri)
            {
                <tr>
                    <td>@libro.Titolo</td>
                    <td>@libro.Isbn</td>
                    <td>@libro.Prezzo.ToString("C")</td>
                    <td>@(libro.Attivo ? "Sì" : "No")</td>
                </tr>
            }
        </tbody>
    </table>
}

@code {
    private bool isLoading = true;
    private string? errorMessage;
    private List<LibroListItemDto> libri = [];

    protected override async Task OnInitializedAsync()
    {
        try
        {
            libri = await CatalogoApiClient.GetLibriAsync();
        }
        catch (Exception ex)
        {
            errorMessage = ex.Message;
        }
        finally
        {
            isLoading = false;
        }
    }
}
```

```dockerfile
FROM mcr.microsoft.com/dotnet/sdk:10.0 AS build
```


```yaml
      - main

permissions:
  contents: read

jobs:
  build-test:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup .NET
        uses: actions/setup-dotnet@v4
        with:
          dotnet-version: '10.0.x'

      - name: Restore
        run: dotnet restore MiniEditore.sln

      - name: Build
        run: dotnet build MiniEditore.sln -c Release --no-restore

      - name: Test
        run: dotnet test MiniEditore.sln -c Release --no-build
```

La pipeline mostra:

```text
build automatica
test automatici
verifica a ogni push e pull request
```

In una versione successiva si può aggiungere:

```text

build Docker image
scansione vulnerabilità
deploy
```
